# Yakka Dee 台词本（Scriptbook）

基于 BBC 儿童英语启蒙动画 **Yakka Dee** 第一季前 15 集（Banana / Dog / Book / Boots / Bike / Duck / Cup / Bus / Peas / Hat / Worm / Boat / House / Lion / Apple）
制作的「台词本」。每一句台词都配有：

- **中文翻译**
- **该句对应的视频截图**
- 可一键 **关闭图片** 或 **关闭翻译** 的开关

适合英语启蒙跟读、亲子共学，也方便对照原声画面理解语义。

## 🚀 使用方法

直接用浏览器打开 **`html/index.html`** 即可。

- 点击 **🖼️ 图片** 按钮：隐藏 / 显示所有截图
- 点击 **🔤 翻译** 按钮：隐藏 / 显示所有中文翻译
- 点击剧集卡片进入对应台词本，页面内有「🏠 总目录」返回首页

> 个别浏览器对本地 `file://` 加载图片有限制，建议用本地静态服务器打开：
>
> ```bash
> python3 -m http.server 8000
> # 然后访问 http://localhost:8000/html/
> ```

## 🛠 制作流程（概述）

1. **抓字幕**：从字幕源站解析每集时间轴与英文原文。
2. **下视频**：下载对应集视频（HLS 源）。
3. **截帧**：用 `ffmpeg` 按每句时间码截帧，生成逐句截图。
4. **翻译**：用 `dict_zh.py` 做中译（精确词典匹配 + 歌曲 `#` / 全大写音效行的兜底逻辑）。
5. **生成页面**：输出自包含 HTML，含图片 / 翻译开关。

> 如需扩充更多剧集，可复用以上流程：补齐字幕 JSON 与截图后，套用相同的 HTML 模板即可。
